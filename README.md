# LQR控制器项目

基于Python实现的线性二次型调节器（LQR）控制器，包含终端可视化和参数对比功能。

## 项目简介

本项目实现了一个完整的LQR控制器仿真系统，具有以下特点：

- **核心功能完整**：实现了LQR算法、系统仿真、终端可视化和参数对比等核心功能
- **用户友好**：提供了交互式命令行界面，操作简单直观
- **可视化效果**：在终端中展示了清晰的状态轨迹和控制输入
- **参数对比**：支持多组Q/R矩阵参数的对比分析，帮助用户理解参数对控制效果的影响
- **可扩展性强**：代码结构模块化，易于扩展新功能和新系统模型

## 技术栈

- Python 3.8+
- 依赖库：numpy、scipy、rich、tabulate

## 功能说明

1. **单组参数运行**：选择系统模型和预设参数组，查看控制效果
2. **多组参数对比**：自动运行所有预设参数组，展示对比结果
3. **用户自定义参数**：输入自定义的Q/R矩阵参数，查看控制效果

## 系统模型

1. **二阶系统**：状态变量为[位置, 速度]，输入为加速度
2. **倒立摆系统**：状态变量为[角度, 角速度, 小车位置, 小车速度]，输入为小车加速度

## 参数说明

- **Q矩阵**：状态权重矩阵，用于惩罚状态误差
- **R矩阵**：控制权重矩阵，用于惩罚控制输入的代价

预设参数组：
- **高状态权重**：Q矩阵较大，R矩阵较小，收敛速度快但控制输入大
- **高控制代价**：Q矩阵较小，R矩阵较大，控制输入平缓但收敛速度慢
- **平衡型**：Q和R矩阵平衡，在收敛速度和控制输入之间取得平衡

## 性能指标

- **收敛时间**：系统状态收敛到稳态的时间
- **稳态误差**：系统达到稳态后的误差
- **最大控制输入**：控制过程中的最大输入值
- **超调量**：系统响应超过稳态值的程度

## 扩展建议

- **自定义系统模型**：修改`lqr_core.py`中的状态空间矩阵
- **添加新功能**：如离散时间LQR、鲁棒性分析等
- **优化性能**：对于大规模系统，可使用更高效的算法
